# GRAMMAR CHECKER

### Introduction

<hr>

A simple grammar checker that checks the grammar of the input text and returns the grammar score, improvements, and reasons.

### Prerequisites

<hr>

#### Dependencies
- FastAPI
- HTMX
- TailwindCSS

#### API
- OpenAI API (Replace with your own API key)
